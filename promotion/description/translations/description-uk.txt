# Application:
Менеджер файлів OI

# Promo text:
Управляйте файлами на карті пам'яті з Менеджером файлів OI File.

# Description:
Менеджер файлів OI дозволяє переглядати вашу SD карту, створювати каталоги, перейменовувати, копіювати, переміщати і видаляти файли. Він також виступає в якості доповнення до інших додатків для відображення «Відкрити» та «Зберегти» діалоги. 

Ця програма вільна від реклами і не вимагає інтернет з'єднання. 

Для повного списку змін та списку найчастіших питань, будь ласка, відвідайте: 
http://www.openintents.org

Ви можете покращити переклад на вашу мову на Launchpad: 
https://translations.launchpad.net/openintents/trunk

Джерельний код цієї вільної та безкоштовної програми доступний на: 
http://code.google.com/p/openintents

Можливості: 
* Показує список файлів з ескізами зображень
* Переміщує, копіює, перейменовує, вилучає файли.
* Створює та вилучає каталоги (теки).
* Надсилає файли по пошті.
* Acts as an extension to other applications, like GMail, to attach files.
* Підтримка розширень багатьох файлів та МІМЕ типів
* Supports PICK_FILE and PICK_DIRECTORY intents so that other applications can use OI File Manager.

Available extensions for "OI File Manager": 
* Show the license information with "OI About".

Applications that work with "OI File Manager": 
* Open and save text files in "OI Notepad".
